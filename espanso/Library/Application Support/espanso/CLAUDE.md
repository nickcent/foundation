# Claude Code Instructions for Espanso Project

This project is managed from both a MacBook Pro and an iMac. Changes made on either machine sync automatically.

## Always Update Snippet Reference

Whenever you make changes to the Espanso snippets in this project, you must update the snippet reference document:

**File to update:** `~/Documents/CENT-VAULT/Espanso/Espanso-Snippet-Reference.md`

This ensures the central reference stays in sync with any new or modified triggers in this repository.

## Sync Setup

- **Espanso (`~/.config/espanso`):** Entire folder synced between MBP and iMac via SyncThing. Use `~` for paths to handle different usernames.
- **Obsidian (CENT-VAULT):** Synced via official Obsidian Sync (paid).
