# foundation

base set of cli/tui tools for GST!!!

## setup

complete following:

0. install [homebrew](https://brew.sh/)
1. `cp .foundation ~/.foundation`
2. `echo source ~/.foundation >> ~/.zshrc`
3. enjoy!

## update

1. `cp .foundation ~/.foundation`
2. enjoy!

no need to repeat sourcing foundation tools from `./zshrc`
